Headings and Paragraphs

  1. Headings are used to create an outline structure on a web page. There are only 6 heading types from h1-h6. For a heading tag we use: <h#>
  2. Paragraphs tags are the workhorse tag of web designs. Most blocks of text use paragraphs for their formatting so remembering this tag is crucial. For a paragraph tag we use <p>. However, if you would like to include a line break use the tag <br> to start a line break.
  3. If you would like a line between the tags use the tag <hr> which is basically just an empty tag. If you would like to preserve multiple spaces or line breaks you can use the preformatted text <pre> tag. This tag is a block level element. Below is an example of what this would look like
