Formatting Tags and Lists

  • HTMl has two types of body element, block level ements and inline elements. Block level elements will automatically appear in sparate lines in your code while inline elements appear horizontally so that the text will not start a new line.
      • Some specific formating tags are...
    • <b> this tag makes text appear bold < /b >
    • <strong> this tag is similar to the bold tag and also makes text appear bold </strong>
    • <i> this tag makes text appear italicized </i>
    • <em> this tag also makes text appear italicized lt/em>
    • <mark> With this tag text will appear highlighted </mark>
    • <small> This tag makes text appear small </small>

    • <ins> this tag makes text appear underlined </ins>
    • <sub> this tag will make text appear as a subscript </sub> For example H2O
    • <sup> this tag makes text appear as a superscript </sup>
