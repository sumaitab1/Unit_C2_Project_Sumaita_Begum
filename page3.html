Comments

In HTML, comments are hidden within the code and do not appear on the webpage. They are useful for developers to add notes or explanations within the code itself. To create a comment in HTML, you must wrap the comment text inside the <!-- comment text --> syntax.

Here is an example of a comment:

  • <!-- This is a comment -->
  • <a href="example.html">Example Link</a>

Attributes

Attributes in HTML are used to provide additional information about an element. They appear inside the opening tag of an element and modify the behavior or appearance of the element. Attributes are written in the format <tagName attributeName="value">, where attributeName is the name of the attribute, and value is the value assigned to it.

Here are some examples:

  • <img src="happyface.jpg">
  • <p style="color:red"> (Which makes text appear red) </p>
