iFrames

  1. iFrames allow one HTML document to display the contents of other HTML documents. We use the <iframe> element to accomplish this. The HTML iframe syntax is: <iframe src="url" title="description"></iframe>
    • The src attribute is used to determine the path to the HTML document page that will provide the contents of the iframe.
  2. To adjust the height and width of the iframe window, we use:
    • <iframe src="url" title="description" height="<b>#</b>" width="<b>%</b>"></iframe>
