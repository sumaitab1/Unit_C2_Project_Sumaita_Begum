Connecting Links into HTML

  • For Links the anchor element is used to insert a hyperlink to another page or location on the same page. Links are considered to be inline elements . To locat the link you need the href attribute (HyperText Reference)
    • <a href ="pathToFile"> Link Text </a>
    • For absolute links we use: <a href="insert link name"> Google </a>
    • For relative or internal links (same folder where web page is located/somewhere in the file structure) we used: <a href= "/index.html"> Home Page </a>
    • To let users download a file we use <a download="filename">
    • To specify a specific plane within the page we use the target tag which is: <a href= "URL" target ="blank"> Blank is just an example of target tags. There is also the _top tag which opens the target link at the top of the page the self tag which opens the link within the iframe and the parent tag which opens the link within the parent of the current frame.
    • You can also embed images (inline element). To locate the image you paste the the link into the src attribute. The full example code for this is < image src="happyface.jpg"alt ="smileyface"> For the alt part of the code, it is a attribute used for visually impaired users. With this attribute screen readers will be able to describe the image to the user making it accessible to all people.
    • You can also adjust the sizing of your image using: < img src="File" width="#" > (You can use width, height or both!)
